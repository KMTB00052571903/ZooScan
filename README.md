# ZooScan

ZooScan es una aplicación web moderna e intuitiva diseñada para facilitar la interacción y el escaneo de información a través de códigos QR. Ofrece una experiencia de usuario fluida con gestión de perfiles, autenticación segura y visualización de estadísticas rápidas.

## 🚀 Características Principales

*   **Escaneo de Códigos QR:** Funcionalidad integrada para interactuar rápidamente con la información.
*   **Autenticación Segura:** Flujos de registro e inicio de sesión.
*   **Gestión de Perfiles:** Visualización y edición de la información del usuario de forma sencilla.
*   **Panel de Estadísticas:** Tarjetas informativas para visualizar datos clave de un vistazo.

## 💻 Tecnologías Utilizadas

Este proyecto está construido con herramientas de desarrollo web de vanguardia para garantizar el mejor rendimiento y una excelente experiencia de desarrollo:

*   **[React 19](https://react.dev/):** Biblioteca principal para construir la interfaz de usuario.
*   **[TypeScript](https://www.typescriptlang.org/):** Añade tipado estático a JavaScript para un código más robusto y fácil de mantener.
*   **[Vite](https://vitejs.dev/):** Herramienta de construcción (bundler) ultrarrápida que proporciona un servidor de desarrollo instantáneo.
*   **[React Router DOM](https://reactrouter.com/):** Manejo de rutas del lado del cliente para una navegación fluida (Single Page Application).
*   **[ESLint](https://eslint.org/):** Herramienta de linting para mantener un estilo de código consistente y libre de errores.

## 🛠️ Instalación y Uso Local

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

### Prerrequisitos

*   [Node.js](https://nodejs.org/) instalado en tu computadora.

### Pasos

1.  Clona este repositorio.
2.  Navega al directorio del proyecto:
    ```bash
    cd ZooScan
    ```
3.  Instala las dependencias necesarias:
    ```bash
    npm install
    ```
4.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```

La aplicación estará disponible por defecto en `http://localhost:5173`.

### Construcción para Producción

Para compilar la aplicación y prepararla para producción, ejecuta:

```bash
npm run build
```
